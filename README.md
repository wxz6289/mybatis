# 项目结构说明

当前仓库按学习主题整理为三个部分：

- `java/`：Java 语言与基础语法、集合、并发等内容
- `mybatis/`：Spring Boot + MyBatis + JDBC 实践项目（数据库与接口相关）
- `spring/`：Spring 体系项目与示例（含 `spring/learn`、`spring/springboot`）

## 整理原则

- 已将原 `learn/` 项目内容迁移为 `mybatis/`。
- 删除了迁移后的旧目录残留，避免同一项目在多个目录重复存在。
- 保留三个主题目录下的现有工程，不做功能性改动。

## 建议使用方式

- 学 Java 基础：进入 `java/`
- 学数据库持久层（MyBatis/JDBC）：进入 `mybatis/`
- 学 Spring 生态：进入 `spring/`
